The **My guilds** page lists the guilds you have access to, and the ones you can register yourself. You can reach it anytime from **My guilds** in the sidebar.

![Access My guilds from the sidebar](/assets/manual/en/guild/sidenav-guilds.png)

![My guilds](/assets/manual/en/guild/guild-list.png)

### My registered guilds

Lists the guilds already registered and configured on RaidOps that you have access to. Click **Enter** to go to the guild's dashboard.

### My unregistered guilds

Lists the Discord servers you administer that aren't fully registered on RaidOps yet:

- **Register** — the RaidOps bot hasn't been invited to this server yet.
- **Complete** — the bot is already invited, but the settings haven't been saved yet.

Both buttons take you to the page described in [Register a guild](/manual/guild/register).

> If you only have access to a single guild that's already set up (and none to register), RaidOps redirects you straight to its dashboard without going through this page.
